## git message 기준💯
  
#### -(마이너스)
-현재 진행중 

#### 0  
-맨 처음 디폴트값(0 이면 복습 안한것)  
  
#### 25
-풀이 보고 푼 것
  
#### 50  
-혼자서 짠 것 (완벽히 이해했다고는 자신 x)  
  
#### 75
-완벽히 이해(여기서 한두번 더 복습하면 100)  
  
#### 100
-거의 다시 볼 필요 없을정도로 완벽 이해
